# Akshat Khanna Portfolio - Enhanced with Basic React JS

This is an enhanced version of the portfolio website using HTML, CSS, JavaScript, and beginner-friendly React JS.

## What was enhanced

- Converted the website into simple React components.
- Added `React JS` to the skills and project tech stack.
- Added a project filter using `useState`.
- Added a dark/light theme button using `useState`.
- Added changing hero text using `useEffect`.
- Added quick stats in the hero section.
- Added a React explanation section so the React work is visible and easy to explain.
- Improved card hover effects, spacing, responsiveness, and visual polish.
- Fixed the profile card structure so the role and quick info stay inside the card.

## React concepts used

- Components
- Props
- `useState`
- `useEffect`
- `.map()` for rendering lists
- Conditional rendering

## Files

- `index.html` - Main HTML file with React CDN links
- `style.css` - Styling and responsive layout
- `script.js` - All React components and simple interactivity
- `assets/` - Put your photo and resume here

## Important assets

Add your existing assets in this folder:

```text
assets/akshat-photo.jpeg
assets/Akshat_Khanna_Resume.pdf
```

The website will still show an `AK` fallback if the photo is missing.

## How to run

Best option: open the folder in VS Code and run it with the Live Server extension.

You can also upload it to GitHub Pages/Netlify. The React CDN links will work online.

## How to explain in an interview

You can say:

> I enhanced my portfolio using basic React JS. I converted repeated UI parts into reusable components, passed data using props, rendered skills and projects using arrays with map, added a project filter using useState, and used useEffect for the changing hero text and scroll animations.
